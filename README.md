<h2 align="center">資料結構 Python實現二叉搜尋樹</h2>



二叉搜尋樹（Binary Search Tree）是一種特殊的二叉樹，又稱為排序二叉樹、有序二叉樹。

二叉搜尋樹具有如下特性：

1. 如果二叉樹的左子樹不為空，則左子樹上所有節點的值均小於它的根節點的值。

2. 如果二叉樹的右子樹不為空，則右子樹上所有節點的值均大於它的根節點的值。

3. 如果獨立地看，左子樹、右子樹也分別為二叉搜尋樹。用遞迴的思想，直到樹的葉子節點。
